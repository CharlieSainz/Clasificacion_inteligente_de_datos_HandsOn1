# Clasificador Lineal — Perceptrón

Este proyecto es un pequeño tutorial en Python sobre cómo funciona el Perceptrón, uno de los clasificadores lineales más básicos dentro del aprendizaje automático.
El notebook fue hecho para entender paso a paso cómo entrena, predice y evalúa un modelo de este tipo usando la librería scikit-learn.

---

## Contenido
- Explicación breve de los fundamentos del Perceptrón  
- Modelo matemático y función de activación  
- Implementación en Python / scikit-learn  
- Pipeline con las etapas:
  - Feature Engineering  
  - Model Selection  
  - Training  
  - Prediction  
  - Evaluation  
- Visualización de la frontera de decisión
